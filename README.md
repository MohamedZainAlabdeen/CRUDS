# CRUDS

A simple web application to perform **Create, Read, Update, Delete, and Search (CRUDS)** operations. Built using HTML, CSS, and JavaScript.

## Features

- dark & light theme
- Add new items
- View a list of items
- Edit existing items
- Delete items
- Search items in real-time
